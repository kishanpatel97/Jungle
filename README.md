# Jungle: An Amazon.com Clone
Clone of Amazon.com with tweaks using React.js, Next.js, NextAuth, Tailwind CSS, and Stripe


# Installation Steps

## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
